# Handoffs

Session handoff documents for NYC Explorer. These capture the state of work when pausing/resuming sessions between AI agents or between AI and Stephen.

## Naming Pattern

`HANDOFF-<topic>-<YYYY-MM-DD>.md`

## What Belongs Here

- Session pause/resume snapshots
- Cross-agent handoff state (ChatGPT → Copilot, Copilot → ChatGPT)
- Context bundles for new sessions

## Archive

Completed handoffs move to `archive/` subfolder.
